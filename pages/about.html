About

Hey! I'm Breno, a web developer and the creator of favPoke. I built this project to practice my skills with HTML, CSS, and JavaScript. I hope you enjoy using it as much as I enjoyed making it!

This site was created using the PokéAPI to fetch the data needed to play the game. Remember that, if abused, the API might refuse to accept your request, resulting in the game breaking. If that happens, reload the page and try again.

Besides the PokéAPI, I also used the repositories for images and icons listed below:

  • Pokemon icons created by Roundicons Freebies – Flaticon
  • Pokemon icons created by Nikita Golubev – Flaticon
